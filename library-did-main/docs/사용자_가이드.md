## 서비스 소개

이 서비스는 **사서가 추천도서를 등록하면, AI가 자동으로 책 소개 영상을 만들어주는** 시스템입니다.

아이들은 키오스크에서 연령대별로 추천도서를 보고, 영상을 시청할 수 있습니다.

---

## 사서용 관리자 페이지

### 접속 방법

1. 웹 브라우저에서 관리자 페이지 주소로 접속
2. 아이디/비밀번호 입력 후 로그인

### 추천도서 등록하기

**[추천도서 관리] 메뉴에서 책을 등록합니다.**

```
┌─────────────────────────────────────────┐
│  📚 추천도서 등록                         │
├─────────────────────────────────────────┤
│                                         │
│  책 제목:  [해리포터와 마법사의 돌    ]   │
│                                         │
│  저자:     [J.K. 롤링                ]   │
│                                         │
│  연령대:   ○ 4-6세                       │
│            ○ 7-9세                       │
│            ● 10-13세  ← 선택             │
│                                         │
│           [ 등록하기 ]                   │
│                                         │
└─────────────────────────────────────────┘
```

**입력 항목:**

| 항목 | 설명 | 예시 |
|------|------|------|
| 책 제목 | 등록할 책의 제목 | 해리포터와 마법사의 돌 |
| 저자 | 책의 저자 이름 | J.K. 롤링 |
| 연령대 | 추천 대상 연령 선택 | 4-6세 / 7-9세 / 10-13세 |

**등록 후 자동으로 일어나는 일:**
1. 책 정보가 저장됩니다
2. AI가 책 소개 영상을 자동으로 생성합니다 (약 3~5분 소요)
3. 완료되면 키오스크에서 바로 볼 수 있습니다

### 영상 생성 현황 확인

**[영상 관리] 메뉴에서 영상 생성 상태를 확인할 수 있습니다.**

```
┌─────────────────────────────────────────┐
│  🎬 영상 현황                            │
├─────────────────────────────────────────┤
│                                         │
│  ✅ 해리포터와 마법사의 돌 - 완료        │
│     J.K. 롤링                           │
│                                         │
│  🔄 마법의 설탕 두 조각 - 생성중         │
│     황선미                              │
│                                         │
│  ⏳ 아기 돼지 삼형제 - 대기중            │
│     영국 전래동화                        │
│                                         │
│  ❌ 빨간 모자 - 실패  [재시도]           │
│     그림 형제                           │
│                                         │
└─────────────────────────────────────────┘
```

**상태 설명:**

| 상태 | 아이콘 | 의미 |
|------|--------|------|
| 대기중 | ⏳ | 영상 생성 순서를 기다리는 중 |
| 생성중 | 🔄 | AI가 영상을 만들고 있는 중 |
| 완료 | ✅ | 영상 생성 완료, 키오스크에서 시청 가능 |
| 실패 | ❌ | 영상 생성 실패, [재시도] 버튼으로 다시 시도 가능 |

### 추천도서 삭제하기

등록한 추천도서를 삭제하려면:
1. [추천도서 관리] 메뉴로 이동
2. 삭제할 책의 [삭제] 버튼 클릭

### 영상 보기

[🎬 영상 보기] 버튼을 누르면 AI가 만든 책 소개 영상이 재생됩니다.

```
┌────────────────────────┐
│         ╳ 닫기         │
│  ┌──────────────────┐  │
│  │                  │  │
│  │    🎬 영상       │  │
│  │                  │  │
│  │      [ ▶ ]       │  │
│  │   재생 버튼 터치   │  │
│  │                  │  │
│  └──────────────────┘  │
│                        │
│  📖 김치가 최고야       │
│  ✍️ 김난지              │
│                        │
└────────────────────────┘
```

**사용 방법:**
1. ▶ 재생 버튼을 터치하면 영상이 시작됩니다
2. 영상이 끝나면 다시보기 버튼이 나타납니다
3. ╳ 닫기 버튼으로 목록으로 돌아갑니다

## 자주 묻는 질문 (FAQ)

### Q: 영상 생성이 실패했어요
**A:** [영상 관리] 메뉴에서 해당 책의 [재시도] 버튼을 눌러주세요.

### Q: 영상이 너무 오래 걸려요
**A:** 보통 3~5분 정도 소요됩니다. 10분 이상 걸리면 [재시도]를 해주세요.

### Q: 등록한 책이 키오스크에 안 보여요
**A:** [영상 관리]에서 해당 책의 상태가 "완료"인지 확인해주세요.

### Q: 연령대를 잘못 선택했어요
**A:** 해당 책을 삭제하고 올바른 연령대로 다시 등록해주세요.
