# 스마트 DID 로컬 설치 가이드 (Windows)

## 시스템 요구사항

- **OS**: Windows 10 Pro 이상
- **CPU**: Intel i3 이상
- **RAM**: 8GB 이상
- **저장공간**: 2GB 이상 여유 공간
- **네트워크**: 인터넷 연결 (설치 시), 도서관 내부망 (운영 시)

---

## 빠른 설치 (3단계)

### 1단계: 프로젝트 복사
USB로 `lib-mvp` 폴더를 PC에 복사 (예: `C:\smart-did\lib-mvp`)

### 2단계: 설치
`scripts\setup.bat` **더블클릭** (관리자 권한 필요)

> ⚠️ "관리자 권한" 확인 창이 뜨면 **예**를 클릭하세요.

설치 중 **메모장**이 자동으로 열리면 아래 항목을 수정하세요:
```
ALPAS_API_URL=http://도서관내부IP/BTLMS/HOMEPAGE/API
ALPAS_API_KEY=도서관에서_받은_API_KEY
ALPAS_LIB_NO=도서관_번호
```
저장 후 메모장을 닫고, 설치 창에서 아무 키나 누르면 계속 진행됩니다.

### 3단계: 실행
`scripts\start-all.bat` **더블클릭**

---

## 상세 설명

### setup.bat이 설치하는 것들
1. **Node.js** - JavaScript 런타임 (자동 다운로드)
2. **Memurai** - Windows용 Redis (영상 생성 큐 관리)
3. **프로젝트 의존성** - npm 패키지들
4. **데이터베이스** - SQLite 초기화
5. **빌드** - 백엔드, 워커, 공유 패키지

설치 시간: 약 5-10분 (인터넷 속도에 따라 다름)

### start-all.bat이 실행하는 것들
| 서비스 | 포트 | 역할 |
|--------|------|------|
| 백엔드 | 3001 | API 서버 |
| 워커 | - | 영상 생성 처리 |
| 프론트엔드 | 5173 | 웹 화면 |

실행 후 자동으로 브라우저가 열립니다.

---

## 키오스크 자동 시작 설정 (선택)

PC 부팅 시 자동으로 서버와 브라우저를 시작하려면:

### 서버 자동 시작
1. `Win + R` → `shell:startup` 입력
2. 시작 프로그램 폴더가 열림
3. `start-all.bat` 파일의 **바로가기**를 이 폴더에 복사

### 브라우저 전체화면 자동 시작
1. Chrome 바로가기 만들기
2. 바로가기 속성 → 대상 끝에 추가:
   ```
   --kiosk http://localhost:5173
   ```
3. 이 바로가기도 시작 프로그램 폴더에 복사

## 문제 해결

### "관리자 권한" 창이 안 뜸
→ `setup.bat` 우클릭 → "관리자 권한으로 실행"

### Node.js 설치 후 재시작 필요
→ PC 재시작 후 `setup.bat` 다시 실행

### "빌드 파일이 없습니다"
→ `setup.bat`을 먼저 실행하세요.

### 서버가 시작되지 않음
1. 포트 3001, 5173이 사용 중인지 확인
2. 방화벽에서 해당 포트 허용
3. `.env` 파일 설정 확인

### ALPAS API 연결 실패
1. 도서관 내부망에 연결되어 있는지 확인
2. `.env`의 `ALPAS_API_URL` 주소 확인
3. API Key가 올바른지 확인

### 영상 생성이 안 됨
1. Memurai(Redis) 서비스가 실행 중인지 확인
   - `서비스` 앱에서 "Memurai" 검색
2. Worker 창이 열려 있는지 확인
3. GEMINI_API_KEY가 설정되어 있는지 확인
